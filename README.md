## 使用方法

1. 联系管理员新增分支(非学术垃圾回收总站人员)/创建自己的分支，向管理员发送分支名称与邮箱（可以发送至微信公众号 Orango）
2. 每日向自己的分支提交，如果在晚上九点之前没有 commit 就会收到提醒邮件；后台会统计连续提交天数以及总计提交天数，当日未 commit 则清零
